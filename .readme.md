## How to use it

Install dependencies

### Backend deps
- cd to backend folder
- run ```npm install```

## Frontend deps
cd to frontend folder
run ```npm install```


- Rename the .envexample to .env and add your MONGO_URI
    - More details are needed about mongo db

## Run Server

```
npm run server
```

## Start Frontend

```
npm run client
```
